A simple and interactive *GenBank Annotation Viewer* built using *Python, Biopython, Streamlit, Pandas, and Plotly*.

This project allows you to upload any .gb, .gbk, or .gbff file and instantly:
- Parse annotated genomic features  
- View genes, CDS, rRNA, tRNA, etc.  
- Explore a clean feature table  
- Generate an interactive genome map
